    Cronograma detallado


    La formulación detallada de un proyecto demanda conocer el tiempo estimado de ejecución de manera general, pero también el tiempo de ejecución de cada una de las actividades.

    Una herramienta gráfica muy útil para diseñar el cronograma son los Gráficos de Gantt, donde se presentan las actividades (etapas del proyecto) y tiempos de ejecución, con el uso de barras horizontales.




    En el cronograma, se relacionan todas actividades que se requieren para la ejecución del proyecto y se muestra gráficamente el tiempo estimado de la ejecución de cada una de ellas (Universidad Nacional, 2007).
